DevOps

Nombre: Winibel Pérez

📌 DevOps es la cultura que une el diseño de software (Dev) con su lanzamiento al mundo
(Ops) para crear tecnología de forma rápida, automática y sin errores.

✨ El Ciclo Infinito:

  1. Planear : Diseñar la idea.
  2. Crear : Escribir el código.
  3. Construir : Unificar el software.
  4. Probar : Verificar que funciona.
  5. Liberar : Preparar el lanzamiento.
  6. Desplegar : Subir a la nube.
  7. Operar : Mantenerlo activo.
  8. Monitorear : Medir el éxito.
